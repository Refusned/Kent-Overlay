# Спецификация онбординга: 30 секунд до работающего Kent

---

## Цель

Пользователь нажимает /start → отвечает на 3 вопроса → получает персонализированного ассистента. Весь процесс < 30 секунд.

Ориентир: Kilo AI обещает "2 минуты до работающего агента". Kent в Telegram может быть в 4 раза быстрее.

---

## Флоу

### Шаг 0: Пользователь нажимает /start

Kent отвечает мгновенно:

```
👔 Привет! Я Kent — ваш AI-сотрудник.

Пишу посты, веду CRM, готовлю документы, анализирую конкурентов. Работаю 24/7 прямо здесь, в Telegram.

Чтобы настроить меня под вас — ответьте на 3 быстрых вопроса. Займёт 30 секунд.
```

**Кнопка:** [Поехали →]

---

### Шаг 1: Имя

```
Как к вам обращаться?
```

Пользователь пишет имя. Kent сохраняет в USER.md.

---

### Шаг 2: Род деятельности

```
{имя}, чем вы занимаетесь?
```

**Inline-кнопки (быстрый выбор):**
- [Предприниматель / ИП]
- [Фрилансер]
- [Маркетолог / SMM]
- [Студент]
- [Другое — напишу]

Kent сохраняет в USER.md + подбирает релевантные КентБайтс.

---

### Шаг 3: Главная задача

```
Какую задачу хотите решить первой?
```

**Inline-кнопки (зависят от роли):**

Если "Предприниматель":
- [Контент для соцсетей]
- [Работа с клиентами / CRM]
- [Документы и отчёты]
- [Анализ конкурентов]

Если "Фрилансер":
- [Ответы клиентам]
- [Коммерческие предложения]
- [Контент для продвижения]
- [Учёт финансов]

Если "Маркетолог / SMM":
- [Контент-план]
- [Посты для соцсетей]
- [Аналитика аккаунтов]
- [Рилс / Shorts]

Если "Студент":
- [Конспекты и подготовка к экзаменам]
- [Курсовая / диплом]
- [Оформление по ГОСТ]
- [Поиск источников]

---

### Шаг 4: Готово! (мгновенная ценность)

Kent сразу выполняет задачу. Не "я готов помочь", а конкретный результат:

```
Готово, {имя}! 🎯

Я запомнил: вы {роль}, и первая задача — {задача}.

Вот что я уже могу сделать прямо сейчас:
[Выполняет мини-версию задачи на основе выбора]

Например, если задача "контент-план":
"Вот набросок контент-плана на эту неделю для {роль}:
Пн — [тема]
Ср — [тема]  
Пт — [тема]

Хотите полный план на месяц? Просто скажите."
```

---

### Шаг 5: Подсказки (через 2 минуты)

Если пользователь молчит 2 минуты после онбординга:

```
Кстати, вот что ещё я умею:

📝 "Напиши пост про..." — генерирую контент
📊 "Проанализируй..." — анализ конкурентов, рынка
📄 "Составь КП для..." — коммерческие предложения
🎯 "Контент-план на месяц" — планирование
📋 "Помоги с документом..." — договоры, акты, счета

Или скажите /рецепты — покажу все готовые сценарии.
```

## Технические требования

### Данные для сохранения (USER.md)

```markdown
# Профиль пользователя

- Имя: {имя}
- Роль: {роль}
- Дата начала: {дата}
- Тариф: триал (до {дата+7})
- Приоритетная задача: {задача}
- Telegram ID: {id}
```

### Inline-кнопки (Telegram Bot API)

- Использовать `InlineKeyboardMarkup` для быстрого выбора
- Callback data: `onboard_role_entrepreneur`, `onboard_role_freelancer`, etc.
- После выбора — убирать кнопки (editMessageReplyMarkup)

### Тайминг

| Шаг | Время | Кумулятивно |
|-----|-------|-------------|
| /start → приветствие | 0.5 сек (ответ бота) | 0.5 сек |
| Приветствие → кнопка "Поехали" | 2 сек (чтение) | 2.5 сек |
| Вопрос 1 (имя) | 5 сек (ввод) | 7.5 сек |
| Вопрос 2 (роль) | 3 сек (кнопка) | 10.5 сек |
| Вопрос 3 (задача) | 3 сек (кнопка) | 13.5 сек |
| Результат | 3 сек (генерация) | 16.5 сек |
| **Итого** | | **~17 сек** |

## Триальная воронка (7 дней)

| День | Действие Kent | Цель |
|------|--------------|------|
| 0 | Онбординг + первый результат | Активация |
| 1 | "Доброе утро, {имя}! Чем помочь сегодня?" | Привычка |
| 3 | "Вы пользуетесь Kent 3 дня. Вот что ещё можно: [3 КентБайта]" | Расширение |
| 5 | "До конца триала 2 дня. За это время Kent помог вам с [статистика]" | Осознание ценности |
| 7 | "Триал завершён. Продолжить работу? [990₽ первый месяц]" + кнопка оплаты | Конверсия |
| 8 | Если не оплатил: "Скучаю! Вот промокод на -20% первого месяца" | Ретенция |

## Метрики онбординга

| Метрика | Цель |
|---------|------|
| % завершивших онбординг (из нажавших /start) | > 70% |
| Среднее время онбординга | < 30 сек |
| % отправивших 2+ сообщение после онбординга | > 50% |
| % активных в день 3 | > 40% |
| % конверсии в платную подписку (день 7) | > 15% |
