## План на первые 4 недели

### Неделя 1 — Запуск

**Пн:** Вводный пост
```
Привет! Это канал Kent — AI-сотрудника в Telegram.

Kent — не чат-бот и не "ещё одна нейросеть". Это ассистент, который:
- пишет контент-планы
- ведёт CRM
- готовит документы
- анализирует конкурентов
- работает 24/7

Здесь буду публиковать рецепты, кейсы и обновления.

Попробовать бесплатно (7 дней): @kent_ai_bot
```

**Ср:** Рецепт — "Контент-план на месяц за 5 минут" (из КентБайтс/smm/01)
- Скриншот результата Kent
- Промпт для копирования
- CTA: "Попробуйте сами →"

**Пт:** Совет
```
Совет дня: как получить от AI максимум

Не пишите: "Сделай контент-план"
Пишите: "Составь контент-план на апрель для кофейни в Казани, 5 постов/неделю, аудитория — девушки 25-35"

Чем конкретнее запрос — тем полезнее результат.
```

### Неделя 2 — Рецепты

**Пн:** Рецепт — "Анализ договора за 3 минуты" (из КентБайтс/lawyers/01)

**Ср:** Кейс — скриншот использования Kent (контент-план / КП / анализ)

**Пт:** Совет — "3 задачи, которые Kent делает лучше ChatGPT" (память, интеграции, автоматизация)

### Неделя 3 — Аудитория

**Пн:** Рецепт — "Коммерческое предложение за 5 минут" (из КентБайтс/freelancers/02)

**Ср:** Сравнение — "Kent vs ChatGPT: 5 отличий" (таблица)

**Пт:** Вопрос-вовлечение
```
А какую рутинную задачу вы бы делегировали AI-ассистенту?

1. Контент-планы и посты
2. Ответы клиентам
3. Документы и отчёты
4. Аналитику
5. Своё (пишите в комментариях)
```

### Неделя 4 — Продажа

**Пн:** Рецепт — "Расчёт налогов ИП" (из КентБайтс/accountants/02)

**Ср:** Отзыв первого клиента (скриншот + цитата)

**Пт:** Промо-пост
```
Итоги месяца: Kent помог X пользователям за Y дней.

Самые популярные задачи:
1. ...
2. ...
3. ...

Хотите попробовать? Первые 7 дней — бесплатно, первый месяц — 990₽.

Начать: @kent_ai_bot
```
