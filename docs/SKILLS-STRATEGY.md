## Новые скиллы: что добавить

### Ярус 1 — Критичные (ставить в v1.1, до первых клиентов)

#### 1. `email-triage` — Триаж почты
**Почему:** #6 по скачиваниям на ClawHub (Gog). Email — первое, что автоматизируют. Kilo AI выделяет email triage как ключевую фичу.

```yaml
---
name: email-triage
emoji: "📬"
triggers:
  - "почта"
  - "проверь почту"
  - "входящие"
  - "дайджест почты"
  - "/mail"
integrations:
  - gog (Gmail)
requires:
  env:
    - GOOGLE_OAUTH_TOKEN
storage: workspace/email-triage/
---
```

**Функционал:**
- Утренний дайджест: категоризация входящих (важные / ответить / информация / спам)
- Автоматические черновики ответов на рутинные письма
- Поиск по почте через естественный язык ("найди письмо от Иванова про договор")
- Cron: каждые 2 часа проверять и уведомлять о важных
- Приоритизация: выделять письма с дедлайнами и запросами на действие

**Целевые роли:** все (предприниматели, фрилансеры, юристы, бухгалтеры)

#### 2. `doc-generator` — Генератор документов
**Почему:** 105 скиллов в категории PDF/Documents на ClawHub. Для российского рынка нужны ИП-специфичные документы, которых нет на ClawHub.

```yaml
---
name: doc-generator
emoji: "📄"
triggers:
  - "документ"
  - "составь документ"
  - "договор"
  - "акт"
  - "счёт"
  - "справка"
  - "/doc"
requires:
  pip:
    - python-docx
    - reportlab
storage: workspace/documents/
---
```

**Функционал:**
- Генерация документов по шаблонам: договор, акт, счёт, КП, ТЗ, доверенность, претензия
- Сохранение реквизитов пользователя (ИНН, ОГРН, адрес, расчётный счёт) — вводишь один раз
- Экспорт в DOCX или PDF
- Подстановка дат, номеров, сумм прописью
- Расчёт НДС, итогов таблиц
- Шаблоны по ГОСТ (для студентов)

**Целевые роли:** предприниматели, юристы, бухгалтеры, фрилансеры

#### 3. `humanize` — Очеловечивание AI-текста
**Почему:** 8 771 скачиваний на ClawHub (#12). Ключевая боль: "текст звучит как от нейросети". Особенно важно для русскоязычного контента.

```yaml
---
name: humanize
emoji: "✨"
triggers:
  - "перепиши человечнее"
  - "убери бото-текст"
  - "сделай живым"
  - "humanize"
  - "/humanize"
---
```

**Функционал:**
- Переписывает AI-сгенерированный текст в естественный стиль
- Убирает маркеры AI-текста: "в заключение хочется отметить", "в современном мире", "нельзя не отметить"
- Адаптация под tone of voice пользователя (деловой / разговорный / экспертный)
- Проверка на AI-детекторы (Turnitin, Антиплагиат)
- Интеграция с social-drafts: автоматическая очеловечивание постов

**Целевые роли:** маркетологи, студенты, контент-мейкеры

#### 4. `finance-tracker` — Финансовый трекер
**Почему:** Нет аналога на ClawHub для российского рынка (QuickBooks/Xero не работают в РФ). Заменяет 1С для микробизнеса.

```yaml
---
name: finance-tracker
emoji: "💰"
triggers:
  - "доход"
  - "расход"
  - "+"
  - "-"
  - "отчёт"
  - "баланс"
  - "/finance"
storage: workspace/finance/
integrations:
  - gog (Google Sheets)
---
```

**Функционал:**
- Быстрый ввод: "+50000 съёмка" или "-3000 аренда"
- Автокатегоризация расходов (реклама, аренда, инструменты, налоги, транспорт)
- Еженедельный/месячный отчёт: доходы, расходы по категориям, прибыль
- Расчёт налогов: УСН 6%, УСН 15%, самозанятый 4-6%, патент
- Напоминания: авансовый платёж через 5 дней, оплата налога до 25-го
- Экспорт в Google Sheets (накопительная таблица)
- Годовой итог для налоговой декларации

**Целевые роли:** ИП, фрилансеры, самозанятые

### Ярус 2 — Важные (добавить в первый месяц)

#### 5. `contract-analyzer` — Анализатор договоров
**Почему:** Одна из самых востребованных задач по КентБайтс. На ClawHub нет ничего для российского права.

```yaml
---
name: contract-analyzer
emoji: "⚖️"
triggers:
  - "проверь договор"
  - "анализ договора"
  - "подводные камни"
  - "/contract"
---
```

**Функционал:**
- Анализ текста договора на 10 ключевых рисков (предмет, цена, штрафы, IP, форс-мажор, расторжение)
- Оценка по 10-балльной шкале каждого пункта
- Ссылки на статьи ГК РФ, ТК РФ, ЗоПП
- Выделение кабальных условий (неустойка >0.5%/день, автопролонгация без уведомления)
- Предложение альтернативных формулировок
- ТОП-3 пункта для переговоров

**Целевые роли:** предприниматели, юристы, фрилансеры

#### 6. `meeting-notes` — Заметки со встреч
**Почему:** Одна из самых скачиваемых категорий. Пересечение: голосовые в Telegram → расшифровка → план действий.

```yaml
---
name: meeting-notes
emoji: "📝"
triggers:
  - "протокол встречи"
  - "заметки со встречи"
  - "что обсудили"
  - "/meeting"
integrations:
  - faster-whisper (STT)
storage: workspace/meetings/
---
```

**Функционал:**
- Принимает аудиозапись встречи (голосовое в Telegram или файл)
- Расшифровка через faster-whisper
- Автоматическое выделение: решения, задачи, дедлайны, ответственные
- Формат вывода: структурированный протокол
- Создание задач в CRM (через crm-notes)
- Напоминания по дедлайнам через cron
- Поиск по прошлым встречам: "что решили по проекту X?"

**Целевые роли:** предприниматели, менеджеры, фрилансеры

#### 7. `competitor-monitor` — Мониторинг конкурентов
**Почему:** Нет аналога на ClawHub. Для русскоязычного рынка — мониторинг VK, Telegram-каналов, Instagram.

```yaml
---
name: competitor-monitor
emoji: "🔍"
triggers:
  - "следи за конкурентом"
  - "мониторинг"
  - "что у конкурентов"
  - "/monitor"
integrations:
  - agent-browser
  - tavily-search
storage: workspace/competitors/
---
```

**Функционал:**
- Добавление конкурентов для отслеживания (ссылка на сайт/соцсеть)
- Еженедельный отчёт: новые посты, изменения цен, акции, новые продукты
- Мониторинг Telegram-каналов конкурентов (контент, частота, вовлечённость)
- Сравнительная таблица: вы vs конкуренты (обновляется автоматически)
- Алерты: конкурент запустил акцию / изменил цены / выпустил новый продукт
- Cron: проверка раз в день

**Целевые роли:** предприниматели, маркетологи

#### 8. `summarizer` — Суммаризатор
**Почему:** #8 на ClawHub (10 956 скачиваний). Универсальный скилл для всех ролей.

```yaml
---
name: summarizer
emoji: "📋"
triggers:
  - "кратко"
  - "суммаризируй"
  - "в чём суть"
  - "пересказ"
  - "/summary"
---
```

**Функционал:**
- Суммаризация текстов любой длины (статьи, PDF, длинные сообщения)
- Суммаризация веб-страниц по ссылке
- Суммаризация PDF-документов (загруженных в Telegram)
- Суммаризация длинных email-цепочек
- 3 формата: ультракраткий (3 тезиса) / стандартный (1 абзац) / развёрнутый (с деталями)
- Выделение ключевых цифр, дат, имён, решений

**Целевые роли:** все

#### 9. `wildberries-ozon` — WB/Ozon помощник
**Почему:** Полностью кастомный. Нет на ClawHub. Wildberries + Ozon = основные маркетплейсы РФ. Огромная аудитория селлеров.

```yaml

  - "ozon"
  - "вб"
  - "озон"
  - "маркетплейс"
  - "/wb"
integrations:
  - agent-browser
  - tavily-search
storage: workspace/marketplace/
---
```

**Функционал:**
- Мониторинг позиций товара по ключевым словам (WB, Ozon)
- Анализ карточки товара: SEO-оценка, рекомендации по улучшению
- Мониторинг отзывов: новые отзывы, негативные (алерт)
- Генерация SEO-описаний для карточек товаров
- Ответы на отзывы покупателей (черновики)
- Мониторинг цен конкурентов на маркетплейсе
- Расчёт юнит-экономики: цена, комиссия, логистика, прибыль

**Целевые роли:** предприниматели (e-commerce), селлеры WB/Ozon

- Расписание публикаций (через cron)

| **github** | GitHub управление | 10 611 | Утилита для coder |

```json

```

Добавить: `"agent-browser"` — используется несколькими кастомными скиллами.
