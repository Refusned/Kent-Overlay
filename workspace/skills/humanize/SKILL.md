# ✨ Humanize — Очеловечивание текста

Переписывает AI-сгенерированный текст так, чтобы он звучал как написанный живым человеком. Для русскоязычного контента.

## Когда активировать

1. Пользователь явно просит: "перепиши человечнее", "сделай живым"
2. Пользователь жалуется: "звучит как робот", "слишком канцелярит"
3. Интеграция с social-drafts: если пользователь настроил автоматическое очеловечивание

## Процесс работы

### 1. Получить текст
Пользователь отправляет текст для обработки или Kent обрабатывает свой только что сгенерированный текст.

### 2. Анализ AI-маркеров
Определить и удалить/заменить:

**Шаблонные конструкции (удалить/переписать):**
- "В современном мире..." → убрать
- "Нельзя не отметить, что..." → убрать
- "В заключение хочется отметить..." → убрать или "Итого:"
- "Давайте разберёмся..." → убрать
- "Важно понимать, что..." → убрать, оставить суть
- "Стоит отметить..." → убрать
- "Как показывает практика..." → конкретный пример
- "Не секрет, что..." → убрать

**Избыточная структурность:**
- Слишком много пронумерованных списков → разбавить абзацами
- Каждый абзац начинается с жирного заголовка → убрать часть
- Одинаковая длина абзацев → варьировать

**Канцеляризмы:**
- "осуществлять" → "делать"
- "производить оплату" → "платить"
- "в рамках" → убрать или упростить
- "на сегодняшний день" → "сейчас"
- "является" → "это" или перестроить предложение
- "данный" → "этот" или убрать
- "вышеуказанный" → убрать

**Эмоциональная стерильность:**
- Добавить 1-2 разговорных выражения (уместных для контекста)
- Добавить короткие предложения среди длинных
- Добавить вопросительные предложения (вовлечение)
- Добавить конкретные примеры вместо абстракций

### 3. Адаптация под стиль

Определить нужный стиль на основе контекста:

| Контекст | Стиль | Пример |
|----------|-------|--------|
| Пост в Instagram | Разговорный, с эмодзи | "Ребят, расскажу..." |
| Telegram-канал | Экспертный, но живой | "Вот штука, которая..." |
| Деловое письмо | Деловой, без канцелярита | "Добрый день, Алексей..." |
| Статья на VC.ru | Экспертный с историями | "Когда мы запускали..." |
| Курсовая работа | Научный, но читаемый | Без изменений (ГОСТ) |

### 4. Выдать результат

```
✨ Очеловеченная версия:

[переписанный текст]

Что изменено:
- Убрано 3 шаблонных конструкции
- Заменено 5 канцеляризмов
- Добавлен разговорный элемент в начале
- Укорочены 2 длинных предложения
```

## Режимы

### Лёгкий (по умолчанию)
- Убрать явные AI-маркеры
- Упростить канцеляризмы
- Сохранить структуру и смысл
- ~80% текста остаётся

### Глубокий (по запросу "перепиши сильнее")
- Полная переработка стиля
- Добавить авторский голос
- Перестроить структуру
- ~50% нового текста

### Антиплагиат (по запросу "для антиплагиата")
- Перефразирование с сохранением смысла
- Замена синонимами
- Изменение структуры предложений
- Цель: уникальность >85% по Антиплагиат/Turnitin

## Интеграция с другими скиллами

- **social-drafts**: после генерации поста предложить "✨ Очеловечить?"
- **doc-generator**: НЕ применять к юридическим документам (формальный стиль нужен)
- **broadcast-composer**: предложить для рассылок

## Ограничения

- НЕ очеловечивать юридические документы, договоры, официальные письма
- НЕ очеловечивать научный текст (если пользователь не попросил)
- НЕ менять фактическую информацию (цифры, даты, имена)
- НЕ добавлять ложную информацию для "живости"

## Обработка ошибок

- Текст не предоставлен → "Отправьте текст, который нужно очеловечить. Или скажите 'последний' — обработаю мой предыдущий ответ."
- Текст слишком короткий (<50 символов) → "Текст слишком короткий для обработки. Отправьте более развёрнутый вариант."
- Текст уже живой → "Текст уже звучит естественно! Если хотите, могу подправить конкретные моменты."
