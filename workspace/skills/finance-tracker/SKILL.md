# 💰 Finance Tracker — Финансовый трекер

Учёт доходов и расходов для ИП, самозанятых и фрилансеров. Замена 1С и Excel для микробизнеса. Всё через Telegram.

## Быстрый ввод

Пользователь пишет в свободной форме:

| Ввод | Как Kent интерпретирует |
|------|------------------------|
| `+50000 съёмка свадьбы` | Доход: 50 000, категория: услуги, описание: съёмка свадьбы |
| `-3000 аренда студии` | Расход: 3 000, категория: аренда, описание: аренда студии |
| `доход 120000 от ООО Ромашка` | Доход: 120 000, источник: ООО Ромашка |
| `расход реклама 15000 таргет вк` | Расход: 15 000, категория: реклама, описание: таргет ВК |
| `+80к консультация` | Доход: 80 000, категория: услуги |

**Правила парсинга:**
- `+` или "доход" → доход
- `-` или "расход" → расход
- "к" после числа = тысячи (80к = 80 000)
- Категория определяется автоматически по ключевым словам

## Категории расходов

| Категория | Ключевые слова |
|-----------|---------------|
| Реклама | реклама, таргет, продвижение, маркетинг, блогер |
| Аренда | аренда, офис, коворкинг, студия, помещение |
| Инструменты | подписка, софт, домен, хостинг, сервис, лицензия |
| Транспорт | такси, бензин, метро, доставка, транспорт |
| Связь | телефон, интернет, связь, СИМ |
| Налоги | налог, УСН, взнос, страховой, ПФР, ФОМС |
| Материалы | материал, закупка, товар, сырьё |
| Зарплата | зарплата, оплата работы, подрядчик, фрилансер |
| Прочее | всё остальное |

Пользователь может указать категорию явно: "-5000 подписка Figma (инструменты)"

## Отчёты

### Команда "отчёт" (по умолчанию — за текущий месяц)

```
💰 Финансовый отчёт за Апрель 2026

📈 Доходы: 285 000 ₽
├ Услуги: 230 000 ₽ (5 операций)
├ Консультации: 40 000 ₽ (2 операции)
└ Другое: 15 000 ₽ (1 операция)

📉 Расходы: 62 400 ₽
├ Реклама: 25 000 ₽
├ Аренда: 15 000 ₽
├ Инструменты: 8 400 ₽
├ Транспорт: 6 000 ₽
└ Прочее: 8 000 ₽

💵 Прибыль: 222 600 ₽
📊 Маржинальность: 78%

💳 Налог (УСН 6%): 17 100 ₽
💳 С учётом вычета взносов: 0 ₽ (взносы покрывают)

📅 С начала года: доходы 840 000 ₽, расходы 198 000 ₽
```

### Команда "отчёт за неделю"
Краткая версия: доходы, расходы, прибыль за последние 7 дней.

### Команда "отчёт за [месяц/квартал/год]"
Развёрнутый отчёт за указанный период.

### Команда "сравни с прошлым месяцем"
Динамика: рост/падение каждой категории в %.

## Расчёт налогов

### Настройка (один раз)
Kent спрашивает при первом использовании:
```
Какая у вас система налогообложения?
1. УСН 6% (доходы)
2. УСН 15% (доходы минус расходы)
3. Самозанятый (НПД 4-6%)
4. Патент
5. ОСНО
```

### Автоматический расчёт
В каждом отчёте:
- Сумма налога за период
- Вычет страховых взносов (для УСН 6%)
- Нарастающий итог за год (для авансовых платежей)
- Дедлайн следующего платежа

### Напоминания
Cron-задачи:
- За 5 дней до дедлайна авансового платежа: "💳 Через 5 дней авансовый платёж по УСН. Сумма: {X} ₽. Дедлайн: {дата}"
- 1 числа каждого месяца: краткий отчёт за прошлый месяц
- Ежеквартально: расчёт авансового платежа

### Фиксированные взносы ИП (2026)
- ОПС: {актуальная сумма} ₽/год
- ФОМС: {актуальная сумма} ₽/год
- 1% с доходов свыше 300 000 ₽
- Дедлайн фиксированных: 31 декабря
- Дедлайн 1%: 1 июля следующего года

## Хранение данных

### Локальное (по умолчанию)
Файл `workspace/finance/ledger.json`:
```json
{
  "settings": {
    "tax_system": "usn6",
    "year": 2026
  },
  "transactions": [
    {
      "id": "tx_001",
      "date": "2026-04-01",
      "type": "income",
      "amount": 50000,
      "category": "services",
      "description": "Съёмка свадьбы",
      "source": "Иванова А."
    }
  ],
  "insurance_paid": 0
}
```

### Google Sheets (опционально)
Если подключён gog → дублировать в Google Sheets:
- Лист "Транзакции": дата, тип, сумма, категория, описание
- Лист "Итоги": месячные итоги
- Лист "Налоги": расчёт по кварталам

## Дополнительные команды

| Команда | Действие |
|---------|----------|
| "удали последнюю запись" | Отменить последнюю транзакцию |
| "покажи все расходы на рекламу" | Фильтр по категории |
| "сколько я заработал в марте" | Доходы за период |
| "топ расходов" | ТОП-5 категорий расходов за месяц |
| "экспорт" | Выгрузить в Google Sheets или CSV |
| "взносы оплачены [сумма]" | Записать уплату страховых взносов |

## Безопасность

- Финансовые данные хранятся ТОЛЬКО локально (workspace/finance/)
- В групповых чатах: скилл НЕ активируется
- Экспорт в Google Sheets — только по явному запросу
- Не отправлять финансовые данные третьим лицам

## Обработка ошибок

- Не удалось распознать сумму → "Не понял сумму. Напишите так: +50000 описание"
- Нет данных за период → "За [период] записей нет. Начните добавлять: +сумма описание"
- Не настроена система налогообложения → "Для расчёта налогов укажите систему: УСН 6%, УСН 15%, самозанятый, патент или ОСНО"

## Ограничения

- Это упрощённый учёт, НЕ замена полноценной бухгалтерии
- Для официальной отчётности в ФНС используйте 1С или специализированный софт
- Kent не подаёт декларации — только рассчитывает суммы
- При сложных ситуациях (ВЭД, несколько ИП, сотрудники) → рекомендовать профессионального бухгалтера
