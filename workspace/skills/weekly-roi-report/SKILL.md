# 📊 Weekly ROI Report — Еженедельный отчёт с ROI

## Когда запускается

1. **Автоматически:** каждый понедельник в 9:00 по часовому поясу клиента (cron: `weekly-report`)
2. **По запросу:** когда клиент спрашивает "что ты сделал за неделю", "отчёт", "статистика"

## Источники данных

Подсчитай метрики из файлов `workspace/memory/YYYY-MM-DD.md` за последние 7 дней. Каждая запись в daily notes имеет формат:

```
### HH:MM — [тип] Содержание
```

Типы для подсчёта:
- `[task]` — выполненная задача
- `[fact]` — записанный факт
- `[decision]` — принятое решение
- `[context]` — контекст разговора

## Что считать

Пройди по daily notes за 7 дней и подсчитай:

| Метрика | Как считать | Оценка времени |
|---------|-----------|---------------|
| **Постов создано** | Количество обращений к social-drafts, content-calendar | ~30 мин на пост |
| **Писем обработано** | Количество обращений к email-triage | ~10 мин на письмо |
| **Документов** | Количество обращений к doc-generator, pptx-manager | ~20 мин на документ |
| **CRM-записей** | Количество обращений к crm-notes | ~5 мин на запись |
| **FAQ-ответов** | Количество обращений к faq-responder | ~5 мин на ответ |
| **Код/скриптов** | Количество обращений к coder | ~15 мин на задачу |
| **Финансовых операций** | Количество обращений к finance-tracker | ~10 мин на операцию |
| **Прочих задач** | Все остальные значимые взаимодействия | ~10 мин на задачу |

Если daily notes не содержат достаточно данных — подсчитай по контексту диалога: сколько раз клиент обращался, какие скиллы использовал.

## Формат отчёта

```
📊 Твоя неделя с Kent (ДД.ММ — ДД.ММ):

✍️ Постов создано: X
📬 Писем обработано: X
📄 Документов: X
📇 CRM-записей: X
❓ FAQ-ответов: X
💻 Задач с кодом: X

⏱ Сэкономлено времени: ~Xч
💰 В рублях (по ставке 1000₽/час): ~X,000₽
📈 Стоимость Kent на этой неделе: X₽

{Если есть что отметить — добавь 1 наблюдение}
Например: "На этой неделе ты чаще всего использовал генерацию постов — 
может, пора составить контент-план на месяц вперёд?"
```

## Правила

1. **Не завышай цифры.** Если за неделю было 3 поста — пиши 3, а не 10. Честность важнее красивых чисел.
2. **Если неделя была тихой** (мало обращений) — не придумывай. Напиши: "На этой неделе было спокойно. Я готов к новым задачам!"
3. **Добавляй одно наблюдение** — что-то полезное, что заметил в паттернах использования.
4. **Ставка для расчёта:** используй 1000₽/час как базовую ставку (средняя для фрилансера). Если в USER.md указана ставка клиента — используй её.
5. **Стоимость Kent на неделе:** раздели месячную подписку на 4. Если не знаешь тариф — используй 6,990₽/мес = ~1,748₽/нед.

## Пример отчёта

```
📊 Твоя неделя с Kent (07.04 — 13.04):

✍️ Постов создано: 8
📬 Писем обработано: 12
📄 Документов: 2 (КП + договор)
📇 CRM-записей: 5
❓ FAQ-ответов: 3

⏱ Сэкономлено времени: ~11ч
💰 В рублях (по ставке 1000₽/час): ~11,000₽
📈 Стоимость Kent на этой неделе: 1,748₽

💡 Заметил, что ты всё чаще просишь посты для Instagram. 
Хочешь — составлю контент-план на апрель?
```
