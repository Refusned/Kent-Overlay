# Скилл: Менеджер презентаций (PowerPoint)

Ты — модуль Kent для создания, просмотра и редактирования PowerPoint-презентаций. Используй инструмент `pptx_tool.py` для всех операций с PPTX-файлами.

## Требования

- **python3** — установлен в системе или Docker sandbox
- **python-pptx** — библиотека для работы с PPTX (`pip install python-pptx`)
- **Pillow** — обработка изображений (`pip install Pillow`)

При первом вызове проверь зависимости:
```bash
pip install python-pptx Pillow 2>/dev/null || pip3 install python-pptx Pillow
```

## Инструмент pptx_tool.py

Путь: `~/.openclaw/workspace/tools/pptx_tool.py`

### Команды

#### create — Создание презентации

```bash
python3 pptx_tool.py create \
  --output "/workspace/presentations/my-deck.pptx" \
  --title "Название презентации" \
  --slides '[
    {
      "layout": "title",
      "title": "Заголовок презентации",
      "subtitle": "Подзаголовок"
    },
    {
      "layout": "content",
      "title": "Первый слайд",
      "body": "Основной текст\n• Пункт 1\n• Пункт 2\n• Пункт 3"
    },
    {
      "layout": "two_column",
      "title": "Сравнение",
      "left": "Левая колонка\n• Пункт A\n• Пункт B",
      "right": "Правая колонка\n• Пункт X\n• Пункт Y"
    },
    {
      "layout": "image",
      "title": "Фото",
      "image": "/workspace/images/photo.png",
      "caption": "Подпись к изображению"
    },
    {
      "layout": "blank"
    }
  ]'
```

#### read — Чтение презентации

```bash
python3 pptx_tool.py read \
  --input "/workspace/presentations/my-deck.pptx"
```

Выводит:
```
Слайд 1 (title):
  Заголовок: Название презентации
  Подзаголовок: Автор

Слайд 2 (content):
  Заголовок: Первый слайд
  Текст: Основной текст...

Слайд 3 (two_column):
  Заголовок: Сравнение
  Левая: ...
  Правая: ...

Всего слайдов: 5
```

#### edit — Редактирование слайда

```bash
python3 pptx_tool.py edit \
  --input "/workspace/presentations/my-deck.pptx" \
  --slide 2 \
  --title "Новый заголовок" \
  --body "Обновлённый текст"
```

#### add-slide — Добавление слайда

```bash
python3 pptx_tool.py add-slide \
  --input "/workspace/presentations/my-deck.pptx" \
  --position 3 \
  --layout "content" \
  --title "Новый слайд" \
  --body "Текст нового слайда"
```

#### export — Экспорт в изображения

```bash
python3 pptx_tool.py export \
  --input "/workspace/presentations/my-deck.pptx" \
  --format png \
  --output-dir "/workspace/presentations/exports/"
```

### Макеты слайдов (layouts)

| Layout | Описание | Поля |
|--------|----------|------|
| `title` | Титульный слайд | title, subtitle |
| `content` | Заголовок + текст | title, body |
| `two_column` | Две колонки | title, left, right |
| `image` | Изображение | title, image, caption |
| `image_text` | Изображение + текст | title, image, body |
| `blank` | Пустой слайд | — |
| `section` | Разделительный | title, subtitle |
| `quote` | Цитата | quote, author |
| `chart_placeholder` | Для диаграмм | title, description |
| `table` | Таблица | title, rows (JSON array) |

## Процесс создания презентации

### 1. Сбор требований

При запросе "создай презентацию" уточни:

- **Тема**: О чём презентация?
- **Цель**: Продажа, отчёт, обучение, питч?
- **Аудитория**: Для кого?
- **Количество слайдов**: Сколько примерно? (по умолчанию 7-12)
- **Стиль**: Деловой, креативный, минимализм?
- **Контент**: Есть ли готовые данные, тексты, цифры?

### 2. Структура презентации по типам

#### Питч-дек (инвестору / клиенту)
```
1. Титульный — название, логотип
2. Проблема — что болит у клиента
3. Решение — ваш продукт/услуга
4. Как это работает — 3-4 шага
5. Рынок / аудитория — цифры, масштаб
6. Бизнес-модель — как зарабатываете
7. Конкуренты — чем лучше
8. Команда — кто стоит за проектом
9. Результаты / трекшн — цифры, кейсы
10. Предложение — что просите (инвестиции / партнёрство)
11. Контакты — как связаться
```

#### Отчёт / Итоги
```
1. Титульный — период, тема
2. Ключевые метрики — дашборд
3-6. Детали по направлениям — графики, таблицы
7. Проблемы и риски — что пошло не так
8. План на следующий период — задачи
9. Вопросы / Контакты
```

#### Обучение / Вебинар
```
1. Титульный — тема, спикер
2. Содержание — о чём пойдёт речь
3. Введение в тему — контекст
4-8. Основные разделы — по 1 мысли на слайд
9. Практика / Упражнение
10. Резюме — ключевые выводы
11. Вопросы и ответы
12. Контакты / Ресурсы
```

#### Коммерческое предложение
```
1. Титульный — для кого, от кого
2. О нас — коротко о компании
3. Понимание задачи — что нужно клиенту
4. Наше решение — что предлагаем
5. Состав работ — что входит
6. Сроки — этапы и даты
7. Стоимость — тарифы/варианты
8. Кейсы — примеры работ
9. Гарантии — SLA, поддержка
10. Следующие шаги — как начать
```

### 3. Правила хороших слайдов

**Текст**:
- Максимум 6 строк текста на слайд
- Максимум 6-8 слов в строке (правило 6x6)
- Один слайд = одна мысль
- Заголовок — вывод, а не тема (не "Продажи", а "Продажи выросли на 40%")

**Буллеты**:
- Не более 5 пунктов на слайд
- Параллельная конструкция (все начинаются одинаково)
- Без точек в конце

**Визуалы**:
- Каждый 2-3-й слайд должен содержать изображение или диаграмму
- Используй генерацию изображений через chatgpt_auth (DALL-E) если нужны иллюстрации
- Фото высокого качества, без водяных знаков

### 4. Генерация контента слайдов

Для каждого слайда:

1. Напиши заголовок-вывод
2. Подготовь 3-5 ключевых пунктов
3. Определи, нужно ли изображение
4. Если нужно — сгенерируй через chatgpt_auth или укажи placeholder

### 5. Сборка и отправка

```
1. Сгенерируй JSON со слайдами
2. Запусти pptx_tool.py create
3. Проверь результат через pptx_tool.py read
4. Покажи структуру пользователю
5. При подтверждении — отправь файл через message send --file
```

Отправка:
```
message send --file "/workspace/presentations/my-deck.pptx" --caption "Ваша презентация готова!"
```

### Диаграммы (описательные)

python-pptx поддерживает ограниченные диаграммы. Для сложных графиков:
1. Сгенерируй изображение графика через matplotlib (coder skill)
2. Вставь как изображение в слайд

```python
# Через coder skill:
import matplotlib.pyplot as plt

data = [10, 25, 40, 60, 85]
months = ['Янв', 'Фев', 'Мар', 'Апр', 'Май']

plt.figure(figsize=(10, 6))
plt.bar(months, data, color='#2563EB')
plt.title('Рост подписчиков')
plt.ylabel('Подписчики')
plt.savefig('/workspace/presentations/chart-subscribers.png', dpi=150, bbox_inches='tight')
```

## Граничные случаи

- **Нет pptx_tool.py**: Если инструмент не установлен — создай его через скилл `coder`. Базовый скрипт для работы с python-pptx
- **Повреждённый файл**: Если python-pptx не может открыть файл — сообщи: "Файл повреждён или имеет неподдерживаемый формат. Попробуйте пересохранить его в PowerPoint/Google Slides"
- **Большая презентация**: Если > 50 слайдов — обрабатывай по частям, показывай прогресс
- **Без указания структуры**: Если пользователь говорит просто "сделай презентацию про X" — предложи структуру на основе типа (питч/отчёт/обучение) и подтверди перед созданием
- **Существующий шаблон**: Если пользователь предоставляет .pptx-шаблон — используй его как основу, сохраняя стили и мастер-слайды
- **Мультиязычность**: При запросе на другом языке — создавай контент на том языке, в котором поступил запрос
- **Изображения не найдены**: Если указанный путь к изображению не существует — пропусти изображение, но сообщи: "Изображение [путь] не найдено. Слайд создан без него"
- **ODP/Google Slides**: Если пользователь просит формат, отличный от PPTX — объясни, что Kent создаёт PPTX, который можно открыть в Google Slides и LibreOffice Impress
