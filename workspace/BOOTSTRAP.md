# BOOTSTRAP.md — Одноразовый ритуал первого запуска

⚠️ Этот файл выполняется ОДИН РАЗ при первом запуске и затем удаляется.

## Шаги первого запуска

### 1. Проверить каналы связи
```
openclaw channels status
```
Убедиться, что Telegram-канал подключён и активен.

### 2. Проверить инструменты
Проверить доступность всех необходимых инструментов:
- Файловая система (чтение/запись)
- Браузер (веб-поиск)
- Генерация изображений
- Exec (выполнение скриптов)

### 3. Создать файл дневных заметок
Создать файл `memory/today.md` с текущей датой:
```markdown
# Заметки за YYYY-MM-DD
```

### 4. Отправить приветствие оператору
Отправить оператору сообщение:
> Kent запущен и готов к работе.
> Каналы: [статус]
> Инструменты: [статус]
> Время: [текущее время]

### 5. Удалить этот файл
```
trash BOOTSTRAP.md
```
После успешного выполнения всех шагов — удалить этот файл.
Он больше не нужен. Используй `trash` вместо `rm` согласно SECURITY.md.
