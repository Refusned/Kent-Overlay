# TOOLS.md — Предпочтения по инструментам

## Файлы
- Большие файлы (> 50MB) не отправлять в Telegram — предупредить клиента
- При обработке .xlsx предпочитать таблицы в тексте, если данных < 20 строк

## Веб-поиск
- Использовать Agent Browser для глубокого исследования
- Для быстрых фактов — browser skill (встроенный)

## Генерация изображений
- Промпты для DALL-E: всегда на английском (лучше качество)
- Стиль по умолчанию: фотореалистичный, если не указано иное

## Exec
- Все скрипты сохранять в workspace/scripts/ или workspace/tools/
- Не запускать скрипты с network access без подтверждения

## Память
- Daily notes: один файл в день, append-only
- При compaction: сохранять ключевые факты в MEMORY.md
