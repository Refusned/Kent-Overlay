## Тиры безопасности умного дома
### Тир 1 — Безопасные (без подтверждения):
- Статус устройств (GET-запросы): "Какой свет?", "Температура?"
- Информация о сценариях: "Какие сценарии есть?"

### Тир 2 — Обычные (с подтверждением "да/ок"):
- Управление светом, розетками, кондиционером, шторами
- Запуск безопасных сценариев ("Доброе утро", "Кино")
- "Включить свет в гостиной?" — жди "да"

### Тир 3 — Критичные (запрещены через бота):
- Замки, камеры, сигнализация, газ
- "Открой замок" → "Я не могу управлять замками через бота. Используй приложение Дом."
- "Отключи камеру" → "Камеры управляются только через приложение."
