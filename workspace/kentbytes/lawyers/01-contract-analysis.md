# Анализ договора: риски за 3 минуты

## Для кого
Юристы, предприниматели — кому нужно быстро проверить договор на подводные камни.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Проанализируй договор на риски.

Тип договора: [аренда / поставка / оказание услуг / подряд / трудовой / другой]
Я выступаю как: [заказчик / исполнитель / арендатор / арендодатель]

Текст договора:
[вставь текст договора или ключевые пункты]

Проверь:
1. **Предмет договора** — чётко ли определён? Нет ли двусмысленности?
2. **Сроки** — реалистичны ли? Есть ли автопролонгация? Условия расторжения?
3. **Цена и порядок оплаты** — фиксированная или плавающая? Когда платить? Штрафы за просрочку?
4. **Ответственность сторон** — симметричная ли? Нет ли кабальных условий?
5. **Штрафы и пени** — размер адекватный? Нет ли несоразмерных неустоек?
6. **Форс-мажор** — есть ли? Достаточный ли перечень?
7. **Интеллектуальная собственность** — кому принадлежат результаты?
8. **Конфиденциальность** — есть ли NDA? Разумные ли сроки?
9. **Порядок разрешения споров** — суд, арбитраж, медиация? В каком городе?
10. **Подводные камни** — мелкий шрифт, ссылки на приложения, неопределённые формулировки

Для каждого пункта:
- Статус: OK / Внимание / Опасно
- Что написано в договоре (кратко)
- Мой комментарий (что это значит простыми словами)
- Рекомендация (оставить / изменить / убрать)

В конце: общая оценка (1-10) и ТОП-3 пункта для переговоров.
```

## Как настроить под себя
- Для предпринимателя: "Объясняй простым языком, без юридических терминов"
- Для юриста: "Ссылки на статьи ГК РФ для каждого замечания"
- "Предложи формулировки для замены проблемных пунктов"

## Пример результата
10 пунктов анализа: 6 OK, 2 Внимание (автопролонгация на 3 года, асимметричные штрафы), 2 Опасно (неустойка 1%/день = 365% годовых, IP переходит заказчику без доплаты). ТОП-3 для переговоров с готовыми формулировками замены.
