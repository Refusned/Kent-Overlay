# Исковое заявление: черновик

## Для кого
Юристы, юрисконсульты, ИП — кому нужно подготовить проект иска.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Tavily Search — рекомендуется (для поиска судебной практики)

## Промпт

```
Подготовь проект искового заявления.

Тип спора: [гражданский / арбитражный / трудовой / о защите прав потребителей]
Суд: [районный суд / арбитражный суд / мировой судья]

Истец:
- [ФИО или наименование, адрес, ИНН]

Ответчик:
- [ФИО или наименование, адрес, ИНН]

Обстоятельства дела:
[Подробно опиши: что произошло, когда, какие документы есть, были ли попытки досудебного урегулирования]

Что требуем:
[например: взыскать задолженность 500 000 руб + проценты + судебные расходы]

Доказательства:
[перечисли имеющиеся: договор, акты, переписка, чеки, экспертиза]

Структура иска (по ГПК/АПК):
1. Шапка (суд, истец, ответчик, цена иска, госпошлина)
2. "ИСКОВОЕ ЗАЯВЛЕНИЕ о [предмет]"
3. Описательная часть (факты в хронологическом порядке)
4. Мотивировочная часть (нормы права + судебная практика)
5. Просительная часть ("ПРОШУ: 1. Взыскать... 2. Взыскать...")
6. Приложения (список по числу лиц)
7. Расчёт цены иска
8. Расчёт госпошлины

Формат: готовый проект для доработки юристом. С расчётами и ссылками на нормы.
```

## Как настроить под себя
- Для арбитража: "Учти обязательный претензионный порядок (30 дней, ст.4 АПК)"
- Для потребителя: "Истец освобождён от госпошлины до 1 млн (ст.333.36 НК)"
- "Добавь ходатайство об обеспечении иска"

## Пример результата
Проект иска: шапка с реквизитами суда и сторон, цена иска 536 000 руб (500к долг + 36к проценты), госпошлина 8 560 руб, описание событий, ссылки на ст.309, 310, 395 ГК РФ, просительная часть, расчёт, 8 приложений. Готов для доработки.

**Важно**: Kent готовит черновик. Финальная версия должна быть проверена квалифицированным юристом.
