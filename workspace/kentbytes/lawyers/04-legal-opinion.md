# Правовое заключение по вопросу

## Для кого
Юристы, юрисконсульты — кому нужно подготовить правовое заключение для клиента или руководства.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Tavily Search — рекомендуется (для поиска судебной практики)

## Промпт

```
Подготовь правовое заключение.

Вопрос: [сформулируй правовой вопрос]
Пример: "Может ли работодатель удержать из зарплаты стоимость испорченного оборудования?"

Контекст:
[Опиши обстоятельства: кто, что произошло, какие документы есть]

Требования:
1. Анализ применимого законодательства (ГК РФ, ТК РФ, КоАП, специальные законы)
2. Ссылки на конкретные статьи с цитированием ключевых норм
3. Судебная практика (если есть релевантные позиции ВС РФ, КС РФ)
4. Выводы и рекомендации
5. Риски каждого варианта действий

Структура:
1. Описание вопроса
2. Правовая квалификация
3. Нормативное обоснование (статьи + краткое содержание)
4. Анализ судебной практики (2-3 релевантных дела)
5. Выводы (да/нет/зависит от обстоятельств)
6. Рекомендации (что делать)
7. Риски (что может пойти не так)

Формат: структурированный документ, 2-4 страницы. Профессиональный юридический язык.
```

## Как настроить под себя
- Для клиента-непрофессионала: "Добавь раздел 'Простыми словами' в начале"
- Для внутреннего использования: "Кратко, без преамбулы, сразу к сути"
- "Подготовь 2 версии: полную и краткую (на 1 страницу)"

## Пример результата
Заключение: вопрос → ст.238, 241, 248 ТК РФ → удержание возможно, но не более среднемесячного заработка, только при наличии приказа в течение месяца → 2 позиции ВС РФ → рекомендация: оформить акт, приказ, уведомить работника → риск: работник оспорит в суде при нарушении процедуры.
