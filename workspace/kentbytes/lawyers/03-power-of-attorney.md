# Доверенность: быстрый шаблон

## Для кого
Юристы, предприниматели, ИП — кому нужно составить доверенность на представителя.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Составь доверенность.

Тип: [простая письменная / нотариальная (подготовь текст для нотариуса)]
Вид: [генеральная / специальная / разовая]

Доверитель:
- ФИО / Наименование: [...]
- Паспорт / ИНН / ОГРН: [...]
- Адрес: [...]

Представитель:
- ФИО: [...]
- Паспорт: [серия, номер, кем выдан, дата]
- Адрес: [...]

Полномочия:
[Опишите, что может делать представитель]
Примеры:
- Представлять интересы в налоговой (ФНС)
- Получать документы в МФЦ
- Подписывать договоры с контрагентами
- Представлять в суде с правом подписи мирового соглашения
- Управлять расчётным счётом в банке

Срок действия: [1 год / 3 года / до дд.мм.гггг / бессрочная]
Право передоверия: [да / нет]

Формат: готовый документ с правильной юридической формулировкой.
```

## Как настроить под себя
- Для налоговой: "Включи полномочия: подача и получение документов, подписание заявлений, получение выписок"
- Для суда: "Все процессуальные полномочия по ст.54 ГПК РФ / ст.62 АПК РФ"
- "Добавь пункт об отзыве доверенности"

## Пример результата
Готовая доверенность: шапка "ДОВЕРЕННОСТЬ", город, дата прописью, данные доверителя и представителя, перечень полномочий, срок, подпись. Для нотариальной — в формате, который принимают нотариусы.
