# Описание для портфолио

## Для кого
Фрилансеры, дизайнеры, разработчики — кому нужно красиво описать свои работы.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Напиши описание проекта для моего портфолио.

Проект: [например: редизайн интернет-магазина детских игрушек]
Клиент: [например: ООО "Карусель" (можно без названия)]
Моя роль: [например: UI/UX дизайнер]
Что сделал: [перечисли основные работы, например: исследование, прототип, дизайн 15 экранов, UI-kit]
Результат: [например: конверсия выросла с 1.2% до 3.8%, время на сайте +40%]
Срок работы: [например: 3 недели]
Инструменты: [например: Figma, Tilda, Adobe Illustrator]

Формат описания:
1. Заголовок (проект + клиент, привлекательно)
2. Задача клиента (1-2 предложения)
3. Моё решение (2-3 предложения, что и как сделал)
4. Результат (цифры, метрики)
5. Технологии / инструменты (список)

Длина: 150-250 слов. Тон: уверенный профессионал, без воды.
Дополнительно предложи:
- Теги / ключевые слова для SEO (5-7 штук)
- Короткую версию (2-3 предложения) для соцсетей
```

## Как настроить под себя
- Для Behance: "Добавь раздел 'процесс' с этапами"
- Для биржи фриланса: "Формат карточки: краткое и продающее"
- Для серии: "Опиши 5 проектов в едином стиле"

## Пример результата
Готовое описание для портфолио: привлекательный заголовок, задача → решение → результат с цифрами. Плюс SEO-теги и короткая версия для соцсетей.
