# Тайм-трекер: учёт часов через Kent

## Для кого
Фрилансеры с почасовой оплатой — кому нужно считать время и выставлять счета.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Веди учёт времени на моих проектах.

Текущие проекты:
1. [например: Логотип для "Солнышко"]
2. [например: Сайт для ИП Петров]
3. [например: SMM для кофейни]

Ставки: [например: дизайн 2000 руб/час, консультация 1500 руб/час]

Команды:
- "старт [проект]" — включить таймер
- "стоп" — остановить, записать
- "+2ч [проект]" — добавить время вручную
- "отчёт" — сводка за неделю
- "счёт [проект]" — итог для клиента

Формат отчёта:
| Проект | Часов | Ставка | Сумма |
|--------|-------|--------|-------|

Веди всё в памяти. Начни с подтверждения.
```

## Как настроить под себя
- "Предупреждай, если на проект потрачено больше [X] часов"
- "Добавляй описание выполненных задач к каждой записи"
- "По воскресеньям присылай сводку за неделю автоматически"

## Пример результата
Команда "отчёт" → таблица: 3 проекта, 28 часов, 62 000 руб. Команда "счёт Солнышко" → текст для клиента: "Отчёт по проекту, 12 часов, задачи: ..., итого: 24 000 руб".
