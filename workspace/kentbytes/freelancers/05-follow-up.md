# Follow-up: не потерять клиента

## Для кого
Фрилансеры, продажники — кому нужно грамотно напомнить о себе клиенту, который "пропал".

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Напиши follow-up сообщение для клиента.

Ситуация: [выбери]
a) Отправил КП, клиент не ответил (прошло 3 дня)
b) Завершил проект, хочу предложить допуслуги
c) Старый клиент, хочу напомнить о себе (прошло 2 месяца)
d) Клиент согласился, но пропал перед оплатой
e) [опиши свою ситуацию]

Контекст:
- Последний контакт: [когда и о чём общались]
- Моя услуга: [что предлагал/делал]
- Что хочу получить: [оплату / новый заказ / обратную связь]

Требования:
1. Ненавязчиво, но с конкретным поводом (не просто "как дела")
2. Короткое (3-5 предложений)
3. С CTA (конкретный вопрос или предложение)
4. 2 варианта: формальный и дружеский
5. Предложи оптимальный канал (email / мессенджер / звонок) и время отправки
```

## Как настроить под себя
- "Составь цепочку из 3 follow-up: через 3 дня, через неделю, через 2 недели"
- "Добавь триггер: если не ответил после 3-го follow-up — финальное сообщение"
- "Используй мягкий юмор, если уместно"

## Пример результата
2 варианта: формальный ("Добрый день, Алексей! Возвращаюсь к нашему обсуждению...") и дружеский ("Привет! Помнишь, мы обсуждали...?"). По 3-4 предложения с конкретным вопросом в конце.
