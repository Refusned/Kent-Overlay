# Коммерческое предложение за 5 минут

## Для кого
Фрилансеры, агентства — кому нужно быстро подготовить КП и выделиться среди конкурентов.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- pptx-manager (скилл) — для оформления в презентацию

## Промпт

```
Составь коммерческое предложение.

Клиент:
- Компания: [например: ООО "Рассвет", производство мебели]
- Задача: [например: разработка сайта-каталога]
- Их боль: [например: текущий сайт на Тильде, нет заявок]
- Бюджет: [например: до 150 000 руб]

Обо мне:
- Услуга: [например: разработка сайтов на Tilda/WordPress]
- Опыт: [например: 40+ проектов за 3 года]
- Кейс: [например: сделал сайт для мебельной фабрики, заявки выросли x3]

Структура КП:
1. Персональное обращение (покажи, что понял задачу)
2. Проблема клиента (их ситуация сейчас)
3. Решение (что сделаю и почему это решит проблему)
4. Этапы работ (3-5 этапов с описанием и сроками)
5. Кейс (до/после с цифрами)
6. Стоимость (пакеты если уместно)
7. Гарантии (что будет если не понравится)
8. Следующий шаг (CTA)

Формат: готовый текст для мессенджера (деловой, но живой).
```

## Как настроить под себя
- Для PPTX: "Оформи как презентацию 8-10 слайдов"
- Для больших проектов: "Добавь раздел 'риски и как их минимизируем'"
- "Включи 3 варианта пакетов: минимальный, оптимальный, максимальный"

## Пример результата
Персонализированное КП: обращение по имени, проблема клиента, решение, 4 этапа с датами, кейс с цифрами, 2-3 пакета, CTA. Готово для отправки.
