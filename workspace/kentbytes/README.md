# КентБайтс — Библиотека рецептов

Готовые промпты и автоматизации для Kent AI Assistant. Каждый рецепт — это проверенный сценарий, который можно скопировать и использовать прямо сейчас.

| [Студенты](students/) | 5 | Студенты вузов, магистранты, аспиранты |

## Формат рецепта

Каждый рецепт содержит:
- **Название** — что делает рецепт
- **Для кого** — целевая аудитория
- **Что нужно подключить** — необходимые интеграции
- **Промпт** — готовый текст для копирования в Kent
- **Как настроить под себя** — подсказки по адаптации
- **Пример результата** — что получится на выходе

## Как использовать

1. Выберите рецепт из нужной категории
2. Скопируйте промпт из блока кода
3. Вставьте в чат с Kent
4. Следуйте инструкциям Kent для настройки под вашу ситуацию
