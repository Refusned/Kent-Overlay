# KPI-дашборд: еженедельный отчёт

## Для кого
Предприниматели, руководители — кому нужен регулярный контроль ключевых метрик бизнеса.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Google Sheets (через Google Drive) — рекомендуется

## Промпт

```
Помоги настроить еженедельный KPI-отчёт для моего бизнеса.

Тип бизнеса: [например: интернет-магазин одежды]
Ключевые метрики, которые я отслеживаю:
- [например: выручка за неделю]
- [например: количество заказов]
- [например: средний чек]
- [например: расходы на рекламу]
- [например: конверсия сайта]

Данные за эту неделю:
- Выручка: [число] руб
- Заказов: [число]
- Средний чек: [число] руб
- Расходы на рекламу: [число] руб
- Посетителей сайта: [число]
- Конверсия: [число]%

Данные за прошлую неделю: [те же метрики или "сравни с целями"]

Задача:
1. Рассчитай динамику каждой метрики (рост/падение в % за неделю)
2. Определи проблемные зоны (метрики ниже цели или с падением)
3. Рассчитай ROAS (возврат на рекламные расходы)
4. Дай 3 конкретных действия на следующую неделю
5. Оформи как краткий отчёт — не больше 1 экрана в Telegram

Формат: компактный, с эмодзи для визуального восприятия (рост, падение, нейтрально).
```

## Как настроить под себя
- Настройте cron-задачу в Kent для автоматического напоминания каждый понедельник
- Добавьте "Сохрани данные в Google Sheets" для накопления истории
- Для e-commerce добавьте: возвраты, повторные покупки, LTV

## Пример результата
Компактный отчёт в формате Telegram-сообщения: 5-7 метрик с динамикой, проблемные зоны выделены, 3 рекомендации. Занимает 1 экран телефона.
