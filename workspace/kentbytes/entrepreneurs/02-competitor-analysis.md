# Анализ конкурентов за 5 минут

## Для кого
Предприниматели, маркетологи — кому нужно быстро понять конкурентное окружение.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Tavily Search — обязательно (для поиска конкурентов)
- Agent Browser — рекомендуется (для анализа сайтов)

## Промпт

```
Проведи конкурентный анализ для моего бизнеса:

Мой бизнес: [например: онлайн-школа английского для детей]
Город/регион: [например: вся Россия, онлайн]
Моя цена: [например: 5000 руб/мес за 8 занятий]

Задача:
1. Найди 5-7 прямых конкурентов (те же услуги, та же аудитория)
2. Найди 3-5 косвенных конкурентов (альтернативные решения)
3. Для каждого конкурента собери: название, сайт, цены, сильные стороны, слабые стороны
4. Составь сравнительную таблицу (конкурент / цена / УТП / слабость)
5. Определи мою потенциальную нишу — где я могу выиграть
6. Дай 3 конкретные рекомендации по позиционированию

Формат: таблица + выводы. Конкретика, не общие фразы.
```

## Как настроить под себя
- Для локального бизнеса укажите город — Kent найдёт местных конкурентов
- Добавьте "Проанализируй их Instagram/VK" для SMM-анализа
- Добавьте "Посмотри отзывы на Яндекс.Картах" для анализа репутации

## Пример результата
Таблица с 8-12 конкурентами, их ценами, УТП и слабостями. Карта позиционирования. 3 конкретных рекомендации вида: "Занять нишу X, потому что конкуренты Y не покрывают Z".
