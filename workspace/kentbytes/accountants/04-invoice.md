# Счёт на оплату за 1 минуту

## Для кого
ИП, бухгалтеры, фрилансеры — кому нужно быстро выставить счёт клиенту.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Подготовь счёт на оплату.

Поставщик (мы):
- Наименование: [ООО "Ромашка" / ИП Иванов Иван Иванович]
- ИНН: [номер]
- КПП: [номер, для ООО]
- Расчётный счёт: [номер]
- Банк: [название]
- БИК: [номер]
- Корр. счёт: [номер]
- Адрес: [юридический адрес]

Покупатель:
- Наименование: [ООО "Василёк" / ИП Петров]
- ИНН: [номер]

Товары/услуги:
[Перечисли: наименование | количество | цена | сумма]
Пример:
- Разработка дизайна логотипа | 1 шт | 25 000 руб
- Разработка брендбука | 1 шт | 40 000 руб

НДС: [без НДС / 20% / 10%]
Основание: [договор №X от дд.мм.гггг / счёт-оферта]

Номер счёта: [авто / указать]
Дата: [сегодня / указать]
Срок оплаты: [5 банковских дней / до дд.мм.гггг]

Оформи счёт в стандартном формате, готовый для отправки по email.
```

## Как настроить под себя
- "Запомни мои реквизиты — в будущем буду указывать только покупателя и сумму"
- "Пронумеруй автоматически (следующий после №[X])"
- "Добавь QR-код для оплаты через СБП" (вручную — Kent подскажет как сгенерировать)

## Пример результата
Готовый счёт: шапка с реквизитами, таблица товаров/услуг, итого с НДС, срок оплаты. Текстовый формат для email. Для PDF-версии — можно попросить оформить через скилл pptx-manager или скопировать в шаблон.
