# Сверка документов: находим расхождения

## Для кого
Бухгалтеры, финансисты, ИП — кому нужно быстро сверить данные между документами.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Google Drive — рекомендуется (для загрузки файлов)

## Промпт

```
Помоги провести сверку документов.

Задача: [например: сверить акт сверки от поставщика с нашими данными]

Данные от контрагента:
[вставь текст или перечисли: дата, номер документа, сумма]

Наши данные:
[вставь текст или перечисли аналогично]

Что сделать:
1. Сравни записи по датам и суммам
2. Найди расхождения (суммы не совпадают, документ есть у них — нет у нас, и наоборот)
3. Оформи таблицу расхождений: дата | документ | их сумма | наша сумма | разница
4. Посчитай итоговое сальдо по каждой стороне
5. Предложи, какие документы запросить для устранения расхождений

Формат: таблица расхождений + итог + рекомендации.
```

## Как настроить под себя
- Для большого объёма: "Данные в Google Sheets — дай мне шаблон, я заполню"
- Для ежемесячной сверки: "Запомни формат — в следующий раз буду присылать только новые данные"
- "Подготовь письмо контрагенту с перечнем расхождений"

## Пример результата
Таблица: 3 расхождения на общую сумму 15 420 руб. Рекомендации: запросить копию счёт-фактуры N123 от 15.03, уточнить сумму по акту N456. Готовый текст письма контрагенту.
