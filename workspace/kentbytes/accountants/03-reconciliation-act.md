# Акт сверки: шаблон за 2 минуты

## Для кого
Бухгалтеры, предприниматели — кому нужно быстро подготовить акт сверки для контрагента.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Подготовь акт сверки взаиморасчётов.

Наша организация: [ООО "Ромашка" / ИП Иванов И.И.]
ИНН: [номер]
Контрагент: [ООО "Василёк" / ИП Петров П.П.]
ИНН контрагента: [номер]
Период: [например: 01.01.2025 — 31.03.2025]
Договор: [номер и дата, если есть]

Операции за период (наши данные):
[Перечисли: дата | документ (счёт/акт/платёжка) | номер | дебет | кредит]
Пример:
- 15.01 | Счёт-фактура | №12 | 50 000 | -
- 20.01 | Платёжное поручение | №45 | - | 50 000
- 01.02 | Счёт-фактура | №28 | 75 000 | -

Сальдо на начало периода: [0 / сумма в пользу кого]

Задача:
1. Оформи акт сверки в стандартном формате (таблица с операциями)
2. Рассчитай сальдо на конец периода
3. Укажи, в чью пользу сальдо
4. Подготовь текст для отправки контрагенту: "Просим подтвердить / сообщить о расхождениях"

Формат: готовый для распечатки текст акта сверки.
```

## Как настроить под себя
- "Оформи в формате, который можно вставить в 1С"
- "Добавь место для подписей и печатей"
- "Подготовь сопроводительное письмо к акту"

## Пример результата
Полный акт сверки: шапка с реквизитами, таблица операций, итоговое сальдо (например: 75 000 руб в пользу ООО "Ромашка"), блок для подписей. Плюс текст письма для отправки.
