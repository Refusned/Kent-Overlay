# Расчёт налогов ИП

## Для кого
ИП, бухгалтеры — кому нужно быстро рассчитать налоги и авансовые платежи.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Рассчитай налоги для ИП.

Система налогообложения: [УСН 6% "доходы" / УСН 15% "доходы минус расходы" / патент / самозанятый]
Период: [квартал / полугодие / год]
Год: [2024 / 2025]

Данные:
- Доходы за период: [сумма] руб
- Расходы за период: [сумма] руб (для УСН 15%)
- Страховые взносы уплачены: [сумма] руб
- Количество сотрудников: [0 / число]
- Авансовые платежи уже уплачены за прошлые кварталы: [сумма] руб

Рассчитай:
1. Налоговую базу
2. Сумму налога
3. Вычеты (страховые взносы — с учётом ограничения 50% для ИП с сотрудниками)
4. Сумму к уплате за текущий период (с учётом ранее уплаченных авансов)
5. Дату дедлайна оплаты
6. КБК и реквизиты для оплаты

Формат: пошаговый расчёт с формулами + итоговая сумма + дедлайн.

Важно: укажи актуальные фиксированные взносы ИП за [год] и лимиты.
```

## Как настроить под себя
- Для квартального расчёта: "Рассчитывай каждый квартал, учитывая нарастающий итог"
- Для самозанятых: "Разбей по ставкам 4% (физлица) и 6% (юрлица)"
- "Напомни за 5 дней до дедлайна оплаты"

## Пример результата
Пошаговый расчёт: доход 850 000 → налог УСН 6% = 51 000 → вычет взносов 45 842 → минус аванс 1кв 10 000 → к оплате: 0 руб (переплата 4 842). Дедлайн: 25 апреля. КБК: 182 1 05 01011 01 1000 110.
