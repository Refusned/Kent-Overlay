# Пояснительная записка к отчётности

## Для кого
Бухгалтеры, финансисты — кому нужно подготовить пояснения к балансу или для налоговой.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Подготовь пояснительную записку.

Тип: [к бухгалтерскому балансу / для налоговой по требованию / к годовой отчётности]
Организация: [наименование]
Период: [год / квартал]

Что нужно пояснить:
[например: значительный рост дебиторской задолженности / убыток за год / расхождение в декларации / причина снижения выручки]

Фактические данные:
- Показатель за текущий период: [сумма/значение]
- Показатель за прошлый период: [сумма/значение]
- Причина изменения: [кратко своими словами]
- Подтверждающие документы: [какие есть]

Требования:
1. Официальный деловой стиль
2. Ссылки на нормативные акты (ПБУ, НК РФ) где уместно
3. Структура: основание → факты → причины → выводы
4. Длина: 1-2 страницы
5. Готово для подписи руководителя

Формат: официальный документ с "шапкой" (кому, от кого, дата).
```

## Как настроить под себя
- Для налоговой: "Тон: максимально нейтральный, факты без оценок"
- Для аудитора: "Добавь ссылки на конкретные строки баланса и первичные документы"
- Для внутреннего использования: "Проще, без формальностей — для директора"

## Пример результата
Официальная пояснительная записка: шапка (генеральному директору ООО "X"), тело с пояснением (причина убытка — инвестиции в оборудование, подтверждение — договоры N1, N2, прогноз — выход на прибыль в Q3), подпись.
