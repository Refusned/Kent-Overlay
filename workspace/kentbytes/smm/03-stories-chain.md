# Сторис-цепочка: вовлечение за 5 слайдов

## Для кого
SMM-менеджеры, блогеры, предприниматели — кому нужны вовлекающие сторис, а не просто "доброе утро".

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Создай сторис-цепочку для Instagram / Telegram.

Тема: [например: как мы выбираем кофейные зёрна]
Цель: [например: показать экспертность + вовлечь аудиторию]
Количество сторис: 5-7
Формат: [текстовые / с фото / с видео / микс]

Для каждой сторис напиши:
1. Номер слайда
2. Визуал (что показать: фото/видео/фон с текстом)
3. Текст на сторис (крупный, 2-3 строки максимум)
4. Подпись (мелкий текст снизу, если нужно)
5. Интерактив (опрос / вопрос / слайдер / тест — где уместно)
6. Стикеры и CTA

Правила:
- Первая сторис = хук (зацепить за 1 секунду)
- Последняя = CTA (что сделать: ответить, перейти, сохранить)
- Минимум 2 интерактивных элемента в цепочке
- Текст короткий, как для чтения на ходу
```

## Как настроить под себя
- Для продающей цепочки: "Последний слайд — оффер с ссылкой"
- Для личного бренда: "Стиль: как будто рассказываю другу"
- Для обучающей: "Формат: факт → миф → правда → совет → вопрос"

## Пример результата
5-7 слайдов с точным описанием визуала, текста и интерактива. Пример: 1) Хук: "90% кофеен покупают зёрна неправильно" (опрос: А вы знали?) → 2) Проблема → 3) Наш подход → 4) Процесс (видео) → 5) CTA: "Хотите попробовать? Напишите в директ".
