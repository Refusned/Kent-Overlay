# Контент-план на месяц за 5 минут

## Для кого
SMM-менеджеры, маркетологи, предприниматели, которые ведут соцсети самостоятельно.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- content-calendar (скилл) — рекомендуется
- Tavily Search — рекомендуется (для трендов)

## Промпт

```
Составь контент-план на месяц для соцсетей.

Бизнес/блог: [например: кофейня в центре Казани]
Площадки: [например: Instagram + Telegram]
Частота публикаций: [например: 5 постов в неделю]
Целевая аудитория: [например: девушки 20-35, любят эстетику, кофе, уют]
Цели: [например: привлечь новых подписчиков, увеличить посещаемость кофейни]
Tone of voice: [например: дружелюбный, уютный, немного ироничный]

Задача:
1. Распредели посты по рубрикам (продуктовые, экспертные, развлекательные, личные, продающие) — оптимальное соотношение для моей ниши
2. Создай календарь на 4 недели: дата → рубрика → тема → формат (пост/карусель/reels/stories)
3. Для каждого поста напиши краткое описание (1-2 предложения о чём)
4. Отметь лучшие дни и время публикаций для моей аудитории
5. Добавь 5 идей для сторис на каждую неделю
6. Предложи 3 хештег-группы (основные, нишевые, локальные)

Формат: таблица (дата | рубрика | тема | формат | описание).
```

## Как настроить под себя
- Для личного бренда: "Добавь рубрику 'за кулисами' и 'мои провалы'"
- Для e-commerce: "Фокус на продуктовых постах, добавь UGC и отзывы"
- Для B2B: "Формат LinkedIn-дружественный, экспертный контент"

## Пример результата
Таблица на 20-22 поста: дата, рубрика, тема, формат. 20 идей для сторис. 3 набора хештегов. Занимает 2-3 экрана в Telegram. Готов для использования.
