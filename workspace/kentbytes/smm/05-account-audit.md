# Аудит аккаунта: что улучшить

## Для кого
SMM-менеджеры, блогеры, предприниматели — кому нужна объективная оценка своих соцсетей.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Agent Browser — рекомендуется (для просмотра профиля)

## Промпт

```
Проведи аудит моего аккаунта в [Instagram / Telegram / VK].

Ссылка: [ссылка на аккаунт]
Ниша: [например: фитнес-тренер]
Цель аккаунта: [например: привлечение клиентов на онлайн-тренировки]

Проанализируй и оцени по 10-балльной шкале:

1. **Шапка профиля** — понятно ли кто я, для кого и что предлагаю?
2. **Визуал** — единый стиль? Качество фото/видео? Привлекает внимание?
3. **Контент-микс** — баланс рубрик (продающие / экспертные / личные / развлекательные)?
4. **Тексты** — читаемость, длина, наличие CTA, хуков?
5. **Вовлечённость** — есть ли вопросы, опросы, интерактив?
6. **Хештеги** — релевантность, количество, микс размеров?
7. **Регулярность** — частота постов, есть ли провалы?
8. **Highlights / Закреп** — структура, актуальность?
9. **Ссылки и CTA** — понятно ли, что делать новому подписчику?
10. **Общее впечатление** — зайду ли я и подпишусь?

Для каждого пункта:
- Оценка (1-10)
- Что хорошо
- Что улучшить (конкретное действие)

В конце: ТОП-5 действий, которые дадут максимальный результат за минимум усилий.
```

## Как настроить под себя
- Для коммерческого аккаунта: "Добавь анализ воронки: от подписки до покупки"
- Для личного бренда: "Оцени, насколько считывается личность"
- Для аккаунта конкурента: "Проведи аудит конкурента [ссылка] и сравни с моим"

## Пример результата
10 пунктов с оценками, конкретными замечаниями и рекомендациями. ТОП-5 быстрых улучшений: переписать шапку, добавить CTA в последние 5 постов, создать 3 highlights, начать задавать вопросы в конце постов, сменить хештег-стратегию.
