# Сценарий Reels / Shorts за 3 минуты

## Для кого
SMM-менеджеры, блогеры, предприниматели — кому нужен сценарий для короткого видео.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Tavily Search — рекомендуется (для трендовых форматов)

## Промпт

```
Напиши сценарий для Reels / YouTube Shorts.

Тема: [например: 3 ошибки при выборе ноутбука]
Ниша: [например: техноблог]
Хронометраж: [30 сек / 60 сек / 90 сек]
Стиль: [говорящая голова / закадровый голос / текст на экране]
Цель: [набрать охваты / продать / обучить]

Формат сценария:
1. ХУК (0-3 сек) — фраза, которая остановит скроллинг
2. ПРОБЛЕМА (3-10 сек) — почему это важно
3. КОНТЕНТ (10-45 сек) — основная информация (3 пункта или история)
4. CTA (последние 5 сек) — что делать зрителю

Для каждой части напиши:
- Текст для озвучки (дословно, как говорить)
- Что показать на экране (визуал / текст / B-roll)
- Таймкод

Дополнительно:
- Предложи 3 варианта хука (от самого провокационного до нейтрального)
- Рекомендуй музыку / тренд, который подойдёт
- Напиши описание для публикации (с хештегами)
```

## Как настроить под себя
- Для продающего: "Добавь мягкую продажу в CTA: ссылка в шапке профиля"
- Для обучающего: "Формат: 'Ты точно не знал, что...'"
- Для развлекательного: "Юмор, мемы, тренды, без продаж"

## Пример результата
Покадровый сценарий: хук "Не покупай ноутбук, пока не посмотришь это" → 3 ошибки с визуалом → CTA "Сохрани, чтобы не забыть". Плюс описание, хештеги, рекомендация по музыке.
