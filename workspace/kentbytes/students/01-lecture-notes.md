# Конспект лекции из аудио/текста

## Для кого
Студенты, слушатели курсов — кому нужно быстро получить структурированный конспект.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- faster-whisper (STT) — для голосовых записей

## Промпт

```
Сделай конспект лекции.

Предмет: [например: макроэкономика]
Тема лекции: [например: инфляция и монетарная политика]

Материал:
[Вставь текст лекции / отправь голосовое сообщение / ссылку на запись]

Формат конспекта:
1. **Заголовок и дата**
2. **Ключевые понятия** (список терминов с определениями)
3. **Основные тезисы** (пронумерованные, 1-2 предложения каждый)
4. **Формулы / модели** (если есть — выделить отдельно)
5. **Примеры** (из лекции)
6. **Связи с другими темами** (что уже проходили)
7. **Вопросы для самопроверки** (5 штук)
8. **Что может быть на экзамене** (ключевые моменты)

Стиль: краткий, по существу. Не пересказ, а структурированная выжимка.
Длина: 1-2 экрана в Telegram.
```

## Как настроить под себя
- Для записи: "Отправлю голосовое — распознай и сделай конспект"
- Для видео: "Вот ссылка на YouTube — сделай конспект" (через youtube-summary)
- "Добавь карточки для Anki (вопрос-ответ) для запоминания"

## Пример результата
Структурированный конспект: 5 ключевых понятий с определениями, 8 тезисов, 2 формулы, 3 примера, 5 вопросов для самопроверки. Занимает 2 экрана. Готов для повторения перед экзаменом.
