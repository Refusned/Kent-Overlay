# Подготовка к экзамену: шпаргалка + тест

## Для кого
Студенты — кому нужно быстро подготовиться к экзамену или зачёту.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Помоги подготовиться к экзамену.

Предмет: [например: гражданское право]
Список вопросов к экзамену:
[вставь список вопросов или темы]

Задача:
1. По каждому вопросу: ответ-шпаргалка (5-7 предложений, только суть)
2. Ключевые определения (выделить жирным)
3. Даты / цифры / формулы (если есть — отдельным блоком)
4. Мнемонические правила для запоминания (где возможно)
5. Связи между вопросами (какие темы пересекаются)

Дополнительно:
- Составь тест на 20 вопросов (4 варианта, 1 правильный) для самопроверки
- Отметь 5 самых вероятных вопросов на экзамене (по опыту)
- Предложи порядок изучения (от простого к сложному)

Формат:
- Шпаргалки: компактные, по 1 экрану на вопрос
- Тест: вопрос → 4 варианта → ответ скрыт (покажу по запросу)
```

## Как настроить под себя
- "У меня 2 дня до экзамена — дай план подготовки по часам"
- "Проведи устный экзамен: задавай вопросы, я отвечаю, ты оцениваешь"
- "Составь карточки формата вопрос-ответ (для Anki)"
- "Объясни тему [X] как для пятиклассника"

## Пример результата
30 шпаргалок по 5-7 предложений каждая. 20 тестовых вопросов с ответами. Топ-5 "горячих" вопросов. План подготовки: день 1 — темы 1-15, день 2 — темы 16-30 + тест.
