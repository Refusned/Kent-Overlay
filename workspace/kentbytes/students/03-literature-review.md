# Литературный обзор: поиск и анализ источников

## Для кого
Студенты, аспиранты — кому нужно найти и систематизировать научные источники.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Tavily Search — рекомендуется
- Agent Browser — рекомендуется (для поиска статей)

## Промпт

```
Помоги с литературным обзором для научной работы.

Тема: [например: "Цифровая трансформация малого бизнеса в России"]
Тип работы: [курсовая / диплом / статья]
Нужно источников: [15-20 / 30-50]

Задача:
1. Найди релевантные источники:
   - Учебники и монографии (3-5)
   - Научные статьи из журналов ВАК / РИНЦ / Scopus (5-10)
   - Диссертации по теме (2-3)
   - Зарубежные источники (3-5)
   - Статистические данные (Росстат, отраслевые отчёты) (2-3)
   - Интернет-источники (2-3)
2. Для каждого источника: автор, название, год, где опубликовано, ключевая идея (1 предложение)
3. Сгруппируй по подтемам
4. Выдели основные точки зрения и противоречия между авторами
5. Предложи структуру параграфа "Обзор литературы" (какие аспекты осветить)

Требования:
- Источники за последние 5-7 лет (кроме классических работ)
- Оформление по ГОСТ Р 7.0.5-2008
- Реальные, проверяемые источники

Формат: список источников + аналитическая записка (1 страница).
```

## Как настроить под себя
- Для аспирантов: "Фокус на зарубежные источники, Scopus/WoS"
- Для курсовой: "Достаточно 15-20 источников, преимущественно на русском"
- "Напиши текст обзора литературы (2-3 страницы), а не только список"

## Пример результата
Список 20 источников по ГОСТ: 4 монографии, 8 статей ВАК, 2 диссертации, 4 зарубежных, 2 статистических. Группировка по 3 подтемам. Записка: ключевые подходы, консенсус, дискуссионные вопросы, пробелы в литературе.
