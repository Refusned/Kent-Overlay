# Оформление по ГОСТ: проверка и исправление

## Для кого
Студенты — кому нужно привести работу в соответствие с требованиями ГОСТ и методички.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно

## Промпт

```
Проверь и исправь оформление моей работы по ГОСТ.

Тип работы: [курсовая / диплом / реферат / отчёт по практике]
Вуз: [если есть специфические требования методички]

Текст для проверки:
[вставь фрагмент или весь текст]

Проверь по ГОСТ:
1. **Список литературы** (ГОСТ Р 7.0.5-2008):
   - Правильный порядок: фамилия, инициалы, название, город, издательство, год, страницы
   - Алфавитный порядок (сначала русские, потом иностранные)
   - Нумерация и форматирование

2. **Оглавление**:
   - Соответствие заголовков в тексте и оглавлении
   - Правильная нумерация разделов и подразделов

3. **Оформление текста**:
   - Сокращения (и т.д., т.е., др.) — допустимы ли?
   - Числительные — когда словами, когда цифрами
   - Кавычки — «ёлочки» для русского текста
   - Тире и дефис — правильное использование

4. **Ссылки и цитаты**:
   - Формат ссылок [1, с.25] или (Иванов, 2023)
   - Все ссылки есть в списке литературы?
   - Нет ли плагиата в цитатах без кавычек?

5. **Таблицы и рисунки**:
   - Нумерация и названия
   - Ссылки в тексте

Для каждой ошибки: что неправильно → как должно быть → правило (ГОСТ/методичка).

Формат: список ошибок + исправленный вариант.
```

## Как настроить под себя
- "Приведи мой список литературы в формат ГОСТ (вот мои источники: ...)"
- "Проверь только список литературы (20 источников)"
- "Используй требования методички [вуза] — вот они: ..."
- "Сделай автоматическое оглавление из заголовков"

## Пример результата
Список ошибок: (1) Источник №3 — неправильный порядок элементов → исправлено, (2) "и т.д." в заголовке — недопустимо → заменить на полную форму, (3) Рис.4 не пронумерован → "Рисунок 4 — Динамика продаж". Исправленный фрагмент текста.
