# План курсовой / дипломной работы

## Для кого
Студенты — кому нужно составить план и структуру научной работы.

## Что нужно подключить
- Базовый Kent (текстовый AI) — обязательно
- Tavily Search — рекомендуется (для поиска источников)

## Промпт

```
Помоги составить план курсовой работы.

Тип работы: [курсовая / дипломная / магистерская диссертация]
Предмет: [например: маркетинг]
Тема: [например: "Влияние SMM на продажи малого бизнеса в России"]
Научный руководитель просит: [если есть конкретные требования]
Объём: [25-30 / 60-80 / 80-120 страниц]

Задача:
1. Сформулируй цель и задачи исследования (3-5 задач)
2. Определи объект и предмет исследования
3. Предложи гипотезу (для дипломной/магистерской)
4. Составь содержание (главы и параграфы с названиями):
   - Глава 1: Теоретическая (обзор литературы, определения, модели)
   - Глава 2: Аналитическая (анализ текущей ситуации, данные)
   - Глава 3: Практическая (рекомендации, проект, расчёты)
5. Для каждого параграфа: 2-3 предложения о содержании
6. Предложи методы исследования (опрос, анализ данных, кейс-стади)
7. Предложи 10-15 источников для литобзора (книги, статьи, сайты)

Формат: готовое содержание + аннотация к каждому разделу.
```

## Как настроить под себя
- Для диплома: "Добавь раздел 'Апробация результатов'"
- Для магистерской: "Акцент на научную новизну и вклад"
- "Подбери источники только за последние 5 лет"
- "Руководитель просит использовать методологию [X]"

## Пример результата
Содержание на 3 главы, 9 параграфов. Цель, 4 задачи, объект/предмет, гипотеза. Аннотация каждого раздела. 12 рекомендованных источников. Готово для утверждения у руководителя.
